_config.yml（整站配置）⭐ 必改
这是你网站的“总设置文件”，里面可以修改：
网站标题 title
描述 description
你的名字、头像
主题颜色 skin
菜单导航栏
社交链接 GitHub、微博、Bilibili 等
插件开关（评论系统等）


index.md 或 index.html（首页内容文件）⭐ 要改

_posts/（你的博客文章）⭐ 可选
里面写文章（Markdown 格式）：
文件名格式必须是：2025-01-01-my-first-post.md

_pages/
存放你网站的页面，例如：
about.md
projects.md
resume.md
你想要“导航栏上有一个「关于我」”，这里创建即可


assets/
用于放：
图片
css 覆盖
自己的 js
例如：
assets/images/avatar.png
